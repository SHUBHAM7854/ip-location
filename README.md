
# IP Location Tracker

Enter an IP and get its Location


## Tech Stack

**Language:** GO, Cobra



## Run Locally

Clone the project

```bash
  git clone https://github.com/SHUBHAM7854/ip-location.git
```

Go to the project directory

```bash
  cd ip-location
```

Run program

```bash
  go build
  ./iptracker trace {{ip_address,DNS_Domain}}
```
